# Axiom Trade Pulse - Complete Setup Guide

## Installation & Setup

```bash
# 1. Create Next.js project
npx create-next-app@14 axiom-pulse --typescript --tailwind --app
cd axiom-pulse

# 2. Install all dependencies
npm install @reduxjs/toolkit react-redux @tanstack/react-query @radix-ui/react-dialog @radix-ui/react-dropdown-menu @radix-ui/react-popover @radix-ui/react-tooltip @radix-ui/react-tabs class-variance-authority clsx tailwind-merge lucide-react recharts

# 3. Install dev dependencies
npm install -D @types/node @types/react @types/react-dom tailwindcss-animate

# 4. Run development server
npm run dev
```
